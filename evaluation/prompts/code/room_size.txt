## Room Size Estimation (m²) from Bounding Boxes

Estimate the **floor area (m²)** of the room shown. If multiple rooms are visible, estimate the **combined** area.

### How to Estimate
1. **Identify room type(s)** (bedroom/living/kitchen, etc.) and whether it’s **one space or multiple connected spaces**.
2. Use bounding boxes to approximate the **walkable/occupied footprint**, not the extreme outermost object boxes.
   - Prefer **large furniture + structural cues** (bed, sofa, counters, closets, walls/doors/windows).
   - **Ignore obvious outliers** (small items, misplaced/isolated boxes) that would unrealistically expand the boundary.
3. Produce a **single numeric estimate** in **m²**.

### Rules
- You **may guess** when uncertain.
- **Never output 0**.
- Keep the reasoning brief (1–3 sentences).

### Output Format (strict)
`{'Reasoning': '...', 'Answer': '<number>'}`
